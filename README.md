# Web Programming Course Portfolio

This repository showcases a collection of projects developed for a Web Programming course, demonstrating various web technologies and concepts.

---

## Projects Overview

### 1. Assignment 2: CSS Animations
**Description:** Exploration of CSS animations featuring two main components: a 3D rotating cube adorned with logos of popular developer platforms (ChatGPT, StackOverflow, LeetCode, etc.) accompanied by background music, and a separate page demonstrating image fade-in/fade-out transitions.
**Technologies:** HTML, CSS

### 2. Assignment 3: Course Preview Page
**Description:** A static web page designed to replicate a Udemy course preview interface. This project demonstrates proficiency in meticulous layout structuring, typography, and responsive design styling.
**Technologies:** HTML, CSS

### 3. Assignment 4: Cuphead Fan Page
**Description:** A dynamic, rich-media fan page dedicated to the video game Cuphead. It features a vast collection of custom character, boss, and map imagery, and utilizes JavaScript for interactive elements, such as dynamic drop-down menus and content showcasing.
**Technologies:** HTML, CSS, JavaScript

### 4. Assignment 5: TV Show Browser
**Description:** An interactive web page designed as a showcase or browser for popular TV shows (e.g., Better Call Saul, Black Mirror, Rick and Morty). It leverages JavaScript to handle dynamic content presentation and user interactions via interactive content boxes.
**Technologies:** HTML, CSS, JavaScript

### 5. Assignment 6: E-commerce Website
**Description:** A comprehensive full-stack e-commerce platform. The backend is built with object-oriented PHP and MySQL, handling product catalogs, categorizations, shopping cart logic, and checkout processes. The frontend is a modern single-page application built with Angular and TypeScript.
**Technologies:** Angular, TypeScript, PHP, MySQL, HTML/CSS

### 6. DOOM Website
**Description:** A tribute webpage dedicated to DOOM, specifically highlighting the "it runs DOOM" internet phenomenon. The page embeds various videos showcasing the game running on unconventional hardware and software (e.g., a pregnancy tester, a PDF, within Minecraft), complete with classic background tracks.
**Technologies:** HTML, CSS

### 7. Snake Game
**Description:** A classic Snake game implementation developed in Java. Configured as a Maven project with web application structures (`src/main/webapp`), indicating a Java-based web application or backend service approach to the classic game.
**Technologies:** Java, Maven
